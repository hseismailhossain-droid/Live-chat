# 💬 Live Chatroom (লাইভ চ্যাটরুম - রিয়েলটাইম কথোপকথন)

একটি সম্পূর্ণ রেসপন্সিভ, প্রিমিয়াম ডিজাইন সমৃদ্ধ এবং রিয়েলটাইম কথোপকথনের চ্যাট প্ল্যাটফর্ম। অ্যাপটিতে রয়েছে ওয়ান-টু-ওয়ান (1-to-1) চ্যাটিং সিস্টেম, রিয়েল-টাইম অনলাইন/অফলাইন উপস্থিতি ট্র্যাকিং, রিড-রিসিপ্টস, টাইপিং সূচক এবং কাস্টম নোটিফিকেশন সাউন্ড।

## 🚀 ১. হোস্টিং গাইড (Hosting Guide) - GitHub থেকে Deployment

এই চ্যাট অ্যাপ্লিকেশনটি চালানোর জন্য ব্যাকএন্ডে **Node.js Express** এবং **WebSockets (WS)** ব্যবহার করা হয়েছে। 

### ⚠️ গুরুত্বপূর্ণ সতর্কতা (Vercel deployment)
* **Vercel** একটি সার্ভারলেস (Serverless) প্ল্যাটফর্ম। সার্ভারলেস ফাংশনে দীর্ঘমেয়াদী ব্যাকএন্ড কানেকশন বা WebSockets লাইভ থাকতে পারে না।
* আপনি যদি অ্যাপটি **Vercel**-এ ডেপ্লয় করেন, চ্যাটিং সেশনটি বার বার বিচ্ছিন্ন (Disconnect) হয়ে যাবে এবং রিয়েলটাইম মেসেজিংয়ে সমস্যা হবে।
* **সুপারিশ (Recommended Deployment):** অ্যাপটি একদম ফ্রিতে এবং ২৫ সেকেন্ডের মধ্যে GitHub থেকে ডেপ্লয় করতে **Render.com** অথবা **Railway.app**-এ হোস্ট করার পরামর্শ দেওয়া হচ্ছে। এরা দীর্ঘজীবী WebSocket প্রসেস সমর্থন করে।

### 🔹 Render.com-এ ফ্রিতে ডেপ্লয় করার নিয়ম:
1. আপনার প্রজেক্টটি GitHub-এ পুশ করুন।
2. [Render.com](https://render.com) এ লগইন করে একটি নতুন **Web Service** সিলেক্ট করুন।
3. আপনার GitHub অ্যাকাউন্ট কানেক্ট করে এই রেপোজিটরি নির্বাচন করুন।
4. নিচের সেটিংসগুলো দিন:
   * **Runtime:** `Node`
   * **Build Command:** `npm run build`
   * **Start Command:** `npm run start`
5. নিচে **Deploy Web Service** এ ক্লিক করুন। আপনার কাজ শেষ! অ্যাপটি লাইভ হয়ে যাবে।

## 🤖 ২. গুগল প্লে-স্টোর (Google Play Store) পাবলিশিং গাইড

এই অ্যাপটিকে আন্ড্রয়েড বা প্লে-স্টোর উপযোগী করতে আমরা ইতিমধ্যেই **Progressive Web App (PWA)** ফিচার যুক্ত করেছি:
* `/public/manifest.json` - প্লে-স্টোর এবং আন্ড্রয়েড সিস্টেম অ্যাপের মেটাডাটা।
* `/public/sw.js` - সার্ভিস ওয়ার্কার ক্যাশিং যা অফলাইন এবং ফাস্ট লোডিং নিশ্চিত করবে।
* `/public/icon-svg.svg` - হাই-ডেফিনিশন অ্যাপ ভেক্টর লোগো।

### 📱 কোড না লিখে প্লে-স্টোর `.aab` বা `.apk` বানানোর নিয়ম:
1. প্রথমে Render বা অন্য হোস্টিং ব্যবহার করে আপনার চ্যাট অ্যাপের লাইভ লিংক তৈরি করুন (যেমন: `https://your-chats.onrender.com`)।
2. মাইক্রোসফটের অফিসিয়াল ও সম্পূর্ণ ফ্রী কনভার্টার [PWABuilder.com](https://www.pwabuilder.com/)-এ যান।
3. আপনার লাইভ লিংকটি সেখানে পেস্ট করুন এবং **"Test My App"** বাটনে ক্লিক করুন।
4. আপনার অ্যাপ স্কোর টেস্ট সফল হলে **"Generate App Bundle"** এ চাপুন।
5. ডাউনলোড করা জিপ ফাইলের ভেতরে আপনি একটি রেডি-টু-আপলোড `.aab` (Android App Bundle) এবং টেস্টিং এর জন্য `.apk` পেয়ে যাবেন।
6. এই `.aab` ফাইলটি দিয়ে আপনি সহজেই আপনার **Google Play Console** এ অ্যাপ পাবলিশ করতে পারবেন!

## 💰 ৩. Adsterra এবং Google AdSense বিজ্ঞাপন (Ads) সেটআপ

আমরা চ্যাটরুমের সবচেয়ে আকর্ষণীয় এবং বেশি ইমপ্রেশন পাওয়ার মতো দুটি জায়গায় রেডিমেড ব্যানার স্লট যুক্ত করে দিয়েছি:
* **লগইন ফর্মে নিচে** ব্যানার স্লট।
* **চ্যাটরুমের খালি অবস্থায়** ব্যাকড্রপ স্লট।

### 🔧 কোড যুক্ত করার ফাইল: `/src/components/AdSlot.tsx`

বিজ্ঞাপন চালু করতে এই ফাইলটিতে যান এবং আপনার পাবলিশার কোড বসিয়ে দিন:

#### 🔹 Google AdSense ব্যবহার করতে:
আপনার AdSense অ্যাকাউন্ট অনুমোদিত হলে `AdSlot` কম্পোনেন্টটি কল করার সময়ে আপনার `adClientId` (ca-pub-xxxx) এবং `adSlotId` পাস করুন:
```tsx
<AdSlot provider="adsense" adClientId="ca-pub-YOUR_ID" adSlotId="YOUR_SLOT_ID" type="banner" />
```

#### 🔹 Adsterra (বা অন্য কোনো ৩য় পক্ষ বিজ্ঞাপন):
আপনার Adsterra পাবলিশার ড্যাশবোর্ড থেকে জাভাস্ক্রিপ্ট অথবা আইফ্রেম কোডটি কপি করুন। তারপর `/src/components/AdSlot.tsx` লাইনে এসে `dangerouslySetInnerHTML` এর ভেতরে থাকা ডামি কোডের জায়গায় আপনার অরজিনাল স্ক্রিপ্টটি বসিয়ে দিন:
```typescript
{
  __html: `<!-- আপনার আসল স্ক্রিপ্ট এখানে দিন -->`
}
```

## 🔍 ৫. গুগল সার্চ কনসোল ও এসইও (Google Search Console & SEO) ভেরিফিকেশন

আপনার অ্যাপটি সবার সামনে গুগলে ফ্রিতে র‍্যাংক করানোর জন্য এবং গুগলের ডেটা ট্র্যাক করার জন্য আমরা সম্পূর্ণ এসইও উপযোগী স্ট্রাকচার এবং ভেরিফিকেশন কনফিগারেশন যুক্ত করে দিয়েছি।

### 🔹১. সাইট ভেরিফিকেশন মেটা ট্যাগ বসানোর নিয়ম:
* আপনার সাইটটি হোস্ট করার পরে **Google Search Console**-এ যান এবং আপনার ডোমেইনটি অ্যাড করুন।
* ভেরিফিকেশন মেথড হিসেবে **HTML tag** অপশনটি বেছে নিন।
* গুগল আপনাকে একটি মেটা ট্যাগ কোড দেবে, যেমন: `<meta name="google-site-verification" content="abcdefgh123456" />`।
* প্রকল্পের `/index.html` ফাইলে যান এবং লাইন ১০ এ থাকা মেটা ট্যাগের `content=""` ভ্যালুটি পরিবর্তন করে আপনার আসল ভেরিফিকেশন কোডটি বসিয়ে দিন:
  ```html
  <meta name="google-site-verification" content="আপনার_আসল_কোড_এখানে_বসবে" />
  ```
* ফাইলটি গিটহাবে পুশ করে দিন এবং সার্চ কনসোলে **Verify** ক্লিক করুন। ইনস্ট্যান্টলি আপনার সাইট ভেরিফাইড হয়ে যাবে!

### 🔹২. সায়টম্যাপ ও রোবটস ডট টিএক্সটি (Sitemap & Robots.txt):
* ইতিমধ্যে আমরা `/public/sitemap.xml` এবং `/public/robots.txt` প্রজেক্টের সাথে যুক্ত করেছি।
* গুগল সার্চ কনসোলের বাম পাশের মেনু থেকে **Sitemaps** সিলেক্ট করুন এবং রুট URL দিয়ে `sitemap.xml` লিখে সাবমিট করুন। যেমন: `your-chats.onrender.com/sitemap.xml`
* গুগল ক্রলার স্বয়ংক্রিয়ভাবে আপনার সাইট ইনডেক্স করা শুরু করবে।
